cd /home - переход в указанную папку;  cd .. - вернуться в предыдущую папку
ls - список файлов в папке; ls -l - с параметрами; ls -la -+скрытые файлы; 
pwd -  вывести рабочий каталог
cp -v (путь к файлу) /(куда копировать) - копирование файла; -v -показывает что скопировалось и куда
(command) --help -помощь по любой команде
man (command) - мануал по команде
mv (путь к файлу) (новый путь)- перенос файлов или перемеинование файлов
cat (путь к файлу) - внутренности файла
less - тоже что cat только открывает в новом окне  
grep что искать - поиск; -i - не учитывать регистр
echo - вывод; echo (текст) > (имя файла) - вставить текст в файл
touch (имена файлов) - создает файлы
mkdir - создание директории
chown (имя пользователя):(группа пользователя) (имя файла) - меняет владельца файла
chmod - изменяет права на файл и может менять права на файл
rm - удаление файла; - r - удаление директории
